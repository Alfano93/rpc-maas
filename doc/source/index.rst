Deployment, setup and installation of Rackspace MaaS for Rackspace Private
clouds. These playbooks allow deployers to monitor clouds using Rackspace
Monitoring as a Service. The playbooks can be used with OpenStack-Ansible
or on their own using Ansible static inventory.
